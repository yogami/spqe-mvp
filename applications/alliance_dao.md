### Application Questions & Humanized Answers

**1. What are you building?**
We are building the first mathematically deterministic hardware firewall for autonomous AI agents. We call it SPQE (Sub-25ms Post-Quantum Enclave).

If you give an AI agent access to a smart contract to execute trades or manage treasuries, you are exposing the entire protocol to prompt-injection attacks. We solve this by operating an AWS Nitro Enclave as a cryptographic co-signer. The agent builds the trade, our Rust enclave isolates the transaction, evaluates the semantic intent in sub-25ms, and either co-signs the transaction or mathematically blocks the wallet drain. It is an intent-based WAF for the blockchain.
